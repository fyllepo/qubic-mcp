# Agent Assignment Guide

Tickets reference which agent type is best suited for the work:

## Agent Types

### `typescript-pro`
Best for: Implementing tools, writing TypeScript code, type safety, testing
**Tickets:** 001-007, 010, 011

### `general-purpose`
Best for: Research, exploration, cross-cutting concerns, CI/CD setup
**Tickets:** 008, 009, 012 (research phase), 013

### `api-designer`
Best for: Designing tool interfaces, schema design, API contracts
**Tickets:** Used as needed when designing new tools

### `backend-developer`
Best for: Server architecture, transport setup, performance optimization
**Tickets:** Used for Phase 5 (remote deployment, Streamable HTTP)

## Working With Tickets

1. Check `backlog/` for available work
2. Move ticket to `in-progress/` when starting
3. Move ticket to `done/` when complete
4. Create new tickets as you discover work
